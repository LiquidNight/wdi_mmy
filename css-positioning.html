By default, all block elements, like divs, have a position: static. When you declare an element something other than static (fixed, absolute, relative), you are creating a coordinate system for the rest of the block elements that follow.
  • Position: static; default - Will not respond to top, bottom, right, left.
  • Position: fixed; means that the position of the box is defined based on the top and left corner of the browser window. Position:fixed means that the div will not move, even if you scroll. It will stay top/left 40px no matter where you scroll.
  • Position: absolute; Similar to fixed, but it will scroll with the page. Absolute positioning is positioned in relation to a parent element that is not static. This box is absolute, in relation to the closest parent element that is not static - the page itself. Therefore, this aboslute box is positioned 40px top/left from the top of the page.
  • Position: relative; Position is moved relative to where it would-have been before.
